# Weather Dashboard 🌦️

A responsive web app that fetches live weather data using the OpenWeather API.

## 🔧 Technologies Used
- HTML
- CSS
- JavaScript
- OpenWeather API

## 📌 Features
- Search any city and get live weather
- Displays temperature, humidity, and condition
- Responsive layout with clean UI
